## 🔍 Prompt 策略分析

### 为什么这个方案有效？

**关键要素：**

1. **明确的规则陈述** ✅
   - 使用"强制执行规则（100%遵守，禁止任何偏离）"这样的强语言
   - 逐条列出具体的执行规则
   - 使用粗体、大写等强调关键词

2. **清晰的角色定义** ✅
   - 将输入分为两部分：**原始小说片段** 和 **修正待检查内容**
   - 明确告诉大模型：前者是上下文，后者是目标
   - 说明上下文的作用："仅供参考语境，不要输出"

3. **防幻觉措施** ✅
   - 明确禁止续写或补全句子
   - 要求保持原有的截断位置
   - 禁止输出解释性文字

4. **格式约束** ✅
   - 使用代码块（```）清晰地界定文本边界
   - 预处理：去除空白字符 + 合并换行符
   - 后处理：使用 `smart_clean()` 清理markdown包裹
