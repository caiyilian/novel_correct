## 🔑 Prompt 策略关键成功因素

1. **强制执行规则** - 明确告诉大模型"100%遵守"
2. **角色清晰** - 分离上下文(context)和核心内容(core)
3. **防幻觉** - 禁止续写、禁止输出解释性文字
4. **格式约束** - 使用代码块清晰界定边界
