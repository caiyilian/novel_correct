## 📊 测试数据分析

### 输入数据特征

```
总长度 = 上下文长度 + 待检查内容长度

推荐配置:
- 总长度: 500-3000 字符
- 上下文 : 待检查内容  = 1:1 到 2:1 的比例
- 避免: 单次处理超过 3000 字符

示例配置 (已验证可行):
- 815 + 509 = 1324 字符 ✅ 
- 69 + 34 = 103 字符 ✅
- 47 + 28 = 75 字符 ✅
```

### 后处理流程

```python
def smart_clean(need_correct, llm_output):
    """
    清除 LLM 返回的 markdown 代码块标记
    同时保持原有的空白习惯
    """
    # 移除 ```
    output = llm_output.strip().strip('`')
    # 保留原始格式
    return output
```
